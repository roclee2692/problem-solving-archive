## 💡 CDQ 分治 - 三维偏序详解

### 核心概念
- **问题**：给定 n 个三维点，对每个点统计有多少个点在三维上都≤它
- **暴力**：O(n²) - 太慢
- **CDQ分治**：O(n log² n) - 快 275 倍！

### 关键技术
- 第一维：排序（预处理）
- 第二维：分治过程中的归并排序
- 第三维：树状数组维护前缀和

### 学习路径
1. 理解树状数组的四个函数：`init`, `lowbit`, `add`, `query`
2. 理解树状数组中每个位置存储的是一个范围的和
3. 理解CDQ分治的三步骤：分治→统计→合并
4. 手动模拟小数据（n=5）的完整分治过程

### 推荐题目
- 洛谷 P3810 - 陌上花开（三维偏序模板题）
- 洛谷 P1975 - 动态逆序对
- BZOJ 3262 - 陌上花开（加强版）
